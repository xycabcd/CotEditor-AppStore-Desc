description

CotEditor это легкий, аккуратный, но мощный текстовый редактор, предназначенный для редактирования простых текстовых файлов, таких как веб-страницы (HTML, CSS), исходные коды программ (Python, Ruby, Perl и т.д.), структурированные тексты (Markdown, Textile, TeX и т. д.) или любые другие виды простых текстов.

Только для Мак

Приложение создано специально для macOS. CotEditor выглядит и работает именно так, как и должно работать приложение в macOS. Это значит, что вы уже знаете как им пользоваться, даже при первом запуске. В нём нет сложных файлов настроек требующих специальных знаний, поэтому вы можете получить доступ ко всем опциям, включая определения синтаксиса и темы, в стандартном окне настроек.

Быстро, просто, удобно

CotEditor запускается так быстро, что вы можете писать текст сразу же когда захотите. Он идеально подходит для написания черновика документа или простого кода. Простой пользовательский интерфейс не отвлекает от работы. Однако CotEditor предназначен не только для новичков и не является типичным минималистичным редактором в стиле «дзен». Он полностью настраиваемый. Конечно же, в нём есть подсветка синтаксиса для разных языков, поиск и замена с помощью регулярных выражений, автоматические отступы, поддержка командной строки и множество других полезных функций. Вы можете легко написать собственный макрос на ваших любимых языках программирования.

Ключевые функции

• Подсветка синтаксиса, более 50 предустановленных языков
• Мощный поиск и замена с помощью регулярных выражений
• Автоматическое резервное копирование файлов документов, с перезаписью или без нее 
• Отображение номеров строк
• Подсчет символов/слов/строк
• Тщательная работа с кодировками файлов и окончаниями строк
• Автоматичекое генерирование содержания для быстрой навигации
• Редактирование с помощью нескольких курсоров и разделение окон редакторов
• Инспектор символов показывающий информацию Unicode
• Режим вертикального расположения текста
• Различные нормализации Unicode
• Полупрозрачные окна
• Поддержка командной строки
• Возможность написания скриптов на вашем любимом языке программирования
• Поддержка и работа с Азиатскими текстами (Иероглифические языки)
